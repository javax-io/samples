### Operators
Operators are special symbols that perform specific operations on one, two, or three operands, and then return a result.

**postfix**

```
++ --
```

**unary**

```
++ -- + - ~ !
```

**multiplicative**

```
* / %
```

**additive**

```
+ -
```

**shift**

```
<< >> >>>
```

**relational**

```
< > <= >= instanceof
```

**equality**

```
== !=
```

**bitwise AND**

```
&
```

**bitwise XOR**

```
^
```

**bitwise OR**

```
|
```

**bitwise NOT**

```
~
```

**logical AND**

```
&&
```

**logical OR**

```
||
```

**logical NOT**

```
!
```

**ternary**

```
? :
```

**assignment**

```
= += -= *= /= %= &= ^= |= <<= >>= >>>=
```
